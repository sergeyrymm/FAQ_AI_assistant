## Раздел 1. Регистрация и вход

### 1.1 Регистрация почты
- Вся почта Mail.ru создаётся с VK ID — единый аккаунт для всех сервисов Mail и VK
- Для регистрации потребуется номер телефона
- Если не приходит СМС с кодом: проверьте правильность номера, попросите повторить отправку, проверьте доступность сети

### 1.2 Вход в почту
**Обычный вход:**
- Через сайт mail.ru
- Через приложение «Почта Mail»

**Вход по коду:**
- Можно войти по коду из СМС или из приложения Почты Mail (для детской почты)
- Код приходит на привязанный телефон

**Вход в другой почтовой программе (Outlook, The Bat!, Thunderbird):**
- Обычный пароль не подойдёт — нужно создать специальный пароль для внешних приложений
- Как создать: Настройки → Все настройки → Безопасность → Пароли для внешних приложений → Создать
- Дать паролю название, выбрать тип протокола, скопировать пароль

**Настройки для почтовых программ (IMAP):**
- IMAP-сервер: imap.mail.ru, порт 993 (SSL/TLS)
- SMTP-сервер: smtp.mail.ru, порт 465 (SSL/TLS)

**Настройки для почтовых программ (POP3):**
- POP3-сервер: pop.mail.ru, порт 995 (SSL/TLS)
- SMTP-сервер: smtp.mail.ru, порт 465 (SSL/TLS)

📄 *Источник: https://help.mail.ru/mail/login/mailer/*

## Раздел 2. VK ID и связка с почтой

### 2.1 Как связать почту с VK ID
**Условия:** Двухфакторная аутентификация должна быть одинаково включена или выключена в обоих аккаунтах

**На сайте:**
- Нажать «Войти с VK ID»
- Ввести пароль от VK ID

**В приложении:**
- Ввести адрес почты → Продолжить
- Нажать «Входить с VK ID»

### 2.2 Ошибка «Нельзя связать эту почту с VK ID»
- Проверьте двухфакторную аутентификацию: она должна быть включена или выключена везде одинаково
- Рекомендуем не отключать 2FA — это защищает ящик от взлома

### 2.3 Вижу чужой VK ID
- Нажмите «Войти в другой аккаунт» и введите свои данные
- Или нажмите «Это не мой аккаунт» — автоматически создастся новый VK ID

### 2.4 Почта связалась с чужим VK ID по ошибке
- Воспользуйтесь инструкцией «Отвязать чужой VK ID»

📄 *Источник: https://help.mail.ru/mail/vkid/link/*

## Раздел 3. Проблемы с письмами

### 3.1 Не приходят письма — пошаговая диагностика
1. Проверьте папку «Спам»
2. Проверьте папку «Корзину»
3. Проверьте, правильно ли отправитель указал ваш адрес
4. Проверьте фильтры (Настройки → Все настройки → Правила фильтрации)
5. Попросите отправителя прислать лог SMTP-сессии или отчёт о недоставке

### 3.2 Все письма приходят в «Спам» или «Корзину»
**Причина:** Настроен фильтр, который помещает туда все письма

**Решение:**
1. Перейдите в Настройки → Все настройки → Правила фильтрации
2. Найдите фильтр с условием: «От» + «не содержит» + любой адрес
3. Удалите такой фильтр

### 3.3 Пропали письма
Возможные причины и решения:
- Удалены автоматически из-за неиспользования (если не заходить в почту больше 6 месяцев, она очищается — восстановить нельзя)
- Удалены вручную или фильтрами
- Находятся в другой папке (Спам, Корзина, социальные сети, рассылки)

### 3.4 Не могу выделить письма
**Новый способ выделения:** Нажимайте на аватарки отправителей (галочка появится на полупрозрачном фоне)

**Выделить все письма в папке:**
- Нажмите «Выделить все» в панели над списком писем
- Или нажмите правой кнопкой мыши на письмо → «Выделить все письма»

**Выделить диапазон:** Выделите первое письмо, затем последнее, удерживая Shift

📄 *Источник: https://help.mail.ru/vkmail/letters/actions/trouble/move/*

## Раздел 4. Пароль и безопасность

### 4.1 Как изменить пароль
**На сайте:**
- Настройки → Безопасность → Пароль → Изменить

**В приложении:**
- Аккаунт → Безопасность (или Вход и безопасность) → Сменить пароль

### 4.2 Если забыл пароль
- Используйте восстановление по номеру телефона
- Код восстановления придёт в СМС

### 4.3 Подозрение на взлом
- Немедленно смените пароль
- Проверьте настройки пересылки писем
- Проверьте последние входы в аккаунт
- При подозрительной активности — обратитесь в поддержку

### 4.4 Управление паролями для внешних приложений
**Удалить пароль:**
1. Настройки → Все настройки → Безопасность → Пароли для внешних приложений
2. Нажмите на нужный пароль → Удалить
3. Подтвердите обычным паролем от почты

📄 *Источник: https://help.mail.ru/mail/login/mailer/*

## Раздел 5. Детская почта

### 5.1 Особенности детской почты
- Нет пароля — вход только по коду из СМС
- Ребёнок входит по коду на телефон, указанный при регистрации
- Взрослый входит через свой ящик, к которому привязана детская почта

### 5.2 Как взрослому войти в почту ребёнка
1. Войдите в свой ящик (к которому привязана детская почта)
2. Нажмите на аккаунт в правом верхнем углу
3. Выберите почту ребёнка из списка

### 5.3 Как ребёнку войти в почту
1. Перейдите на страницу входа
2. Введите адрес детской почты
3. Введите код из СМС (или из приложения Почты Mail)

### 5.4 Ребёнок не может войти
- Проверьте правильность адреса
- Убедитесь, что СМС приходит на тот номер, который был указан при регистрации
- Если нет доступа к телефону — обратитесь в поддержку для смены номера

### 5.5 Не помню, к какому ящику привязана детская почта
- Найдите письмо от Почты Mail с темой «Вы создали детскую почту»
- Детский ящик можно привязать к любому почтовому сервису (не только Mail)

📄 *Источник: https://help.mail.ru/widget/mail/login/kid/*

## Раздел 6. Настройки и личные данные

### 6.1 Как изменить имя, фамилию или никнейм
**Только с компьютера:**
1. Настройки → Все настройки → Аккаунт → Личные данные
2. Заполните поля Имя, Фамилия или Никнейм
3. Сохранить

### 6.2 Как изменить аватарку
**На сайте:**
1. Настройки → Все настройки → Аккаунт → Личные данные
2. Нажмите «Изменить фото»
3. Выберите фото → Открыть → Применить

**В приложении:**
1. Аккаунт → Личные данные → Имя и аватар
2. Нажмите на аватарку → сделайте фото или выберите из галереи

### 6.3 Не меняется аватарка
- Возможно, заблокирована страница в «Моём Мире»
- Напишите в поддержку «Моего Мира» для разблокировки

### 6.4 В письмах отображается не моё имя
- Измените имя по инструкции выше
- Это отдельная настройка, не связанная с подписью в письмах

📄 *Источник: https://help.mail.ru/mail/settings/personal/*

## Раздел 7. Приложение «Почта Mail»

### 7.1 Возможности приложения
- Несколько аккаунтов (можно подключить Яндекс, Gmail, Yahoo и другие)
- Календарь и задачи (планер)
- Видеозвонки (до 100 человек)
- Облачное хранилище
- Голосовой помощник Маруся
- Тёмная тема и фоны
- Пин-код для входа
- Офлайн-режим (письма сохраняются в памяти телефона)
- Группировка писем в цепочки

### 7.2 Быстрые действия с письмом
Проведите по письму справа налево, чтобы увидеть действия:
- Отметить прочитанным
- Поставить флаг
- Переместить
- Удалить

### 7.3 Не открывается приложение
- Очистите кэш приложения
- Проверьте наличие обновлений
- Переустановите приложение

### 7.4 Очистка кэша
- Настройки телефона → Приложения → Почта Mail → Память → Очистить кэш

📄 *Источник: https://help.mail.ru/mail/docs/app/*

## Раздел 10. Переход с другого сервиса

### 10.1 Перенос Gmail на @xmail.ru
**Причина:** Вход в российские сервисы с иностранным ящиком ограничен (ФЗ № 406-ФЗ)

**Как перенести:**
1. Откройте сайт @xmail.ru → Начать
2. Выберите аккаунт Google для переноса
3. Разрешите доступ
4. Укажите номер телефона
5. Подтвердите кодом из СМС
6. Создастся новый VK ID и почта

📄 *Источник: https://help.mail.ru/mail/account/moving/*

## Раздел 11. Обращение в поддержку

### 11.1 Когда обращаться в поддержку
- Если письма нет ни в Спаме, ни в Корзине, и вы уверены, что отправитель его выслал
- При подозрении на взлом
- Если нет доступа к телефону для детской почты
- При блокировке страницы в «Моём Мире»

### 11.2 Что предоставить для ускорения решения
- Лог SMTP-сессии (для проблем с доставкой писем)
- Отчёт о недоставке
- Код ошибки (если есть)

📄 *Источник: https://help.mail.ru/mail/contact/*
